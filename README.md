# ng-node-token-seed

This a seed project which enables basic JWT-based auth in angular with nodejs

## Preconfig

You need to config this before you run it

- nodejs
- MongoDB
- Grunt a tool for build js automaticly
- bower a tool for assign js libiarys

## Run
```
npm install
grunt
npm start
```
